# edf-subtitle-mod

Real-time subtitle overlay for **Earth Defense Force 5** (PC).
Hooks voice playback via Frida and displays subtitles in a Win32 layered window.

**Vibe Coding Warning**: This waaaaaaaagh thing can run thanks to Claude-Sonnet-4.6.

**WIP Warning**: Yeah it works, but is not reaching expectations. Many works needs to be done.

Main problems:
* **So far only zh-tw is covered, you need your own subtitles.txt and cuelength.txt.**
* The thing only works in windowed or borderless (which is not even a choice in-game without proper tool or modification).
* The voicelines are text-generated by *whisper*, and later reviewed by Chatgpt-CODEX with an in-game term dict for ref. **Expect badly translated lines.**
* We use "the funny snake language", as many suggest, it is neither elegant nor graceful. But hey it works.

## Requirements

- EDF5 running in **borderless windowed** mode
- Python 3.11+, [uv](https://github.com/astral-sh/uv)
- Run as **Administrator** (required for Frida to attach)

## Usage

```
uv run python voice_hook.py
```

Launch EDF5 first, then run the script. Press `Ctrl+C` to stop.

## Data files

| File | Format | Purpose |
|------|--------|---------|
| `subtitles.txt` | `cue=subtitle text` | Subtitle text per voice cue |
| `cuelength.txt` | `cue=duration_ms` | Audio duration for display timing |

Cue names ending in `_E` are normalised to the base key before lookup.

## Overlay

- Up to **5 subtitles** stacked simultaneously; oldest-expiring slot evicted when full
- Radio dialogue — yellow (`#FFF0A6`)
- NPC ambient — blue (`#8FE7FF`)
- Click-through, non-activating, anchored to the game monitor
